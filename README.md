# 🎬 Movie Recommendation System

A simple and interactive Movie Recommendation System built using HTML, CSS, and JavaScript. This project suggests movies based on genres, ratings, and other attributes using client-side logic — no backend or database required!

# 📌 Features

- 🎥 List of movies with posters, genres, and ratings
- 🧠 Basic recommendation logic using tags/genres
- 🔍 Search functionality
- 🌟 Highlight top-rated movies
- 💻 Fully responsive design

## 🧰 Tech Stack

 Frontend: HTML, CSS, JavaScript
 IDE: Visual Studio Code
